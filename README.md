# Welcome to keeper

A minimalist note-keeping app using React.

This is a local version of the master branch.

![homepage](images/homepage.png)

## Install

```bash
npm install
```

## Run server

```bash
npm start
```

## Open app

Your app will now be accessible at localhost:3000
